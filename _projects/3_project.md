**RaLiBEV** fuses radar and LiDAR features in a bird's-eye-view representation for robust object detection in adverse weather, using an anchor-box-free detection head.

Published in **IEEE Transactions on Circuits and Systems for Video Technology**, 35(5):4130–4143, 2025 (Y. Yang\*, J. Liu\*, T. Huang, Q.-L. Han, G. Ma, B. Zhu).
